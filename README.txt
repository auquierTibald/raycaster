:Global Controls:

    M : Switch mode ( 3D / Editor )

    X : Shutdown

:Editor controls:

    ARROWS : Editor Camera Movements

    Textures Square :
        LEFT_CLICK : Next texture / Object template
        RIGHT_CLICK : Previous one

    First button : Switch object type ( Walls / Sprites / Animated_Sprite / NPC )

    Second button : Save all the map data in a file ( type filename in the terminal )

    Third button : Load map data from a file ( type path in the terminal )

:3D Controls:

    ZQSD : Movements

    MOUSE : Camera Movements

    E : Interaction ( doors )

    R : Drink 86

:G++ Command:
g++ -Lsrc/lib -Isrc/include main.cpp -o main -lSDL2 -lSDL2_image -LSDL2_ttf
